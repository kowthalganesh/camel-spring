Apache Camel Example using Servlet REST & Web Container

This example shows how to use Servlet REST to define REST endpoints in Camel routes using the Rest DSL
For Java DSL the routes are defined in Java code, in the org.apache.camel.example.rest.RestRouteBuilder class.

There is a provider REST service that supports the following operations

  • GET /rest/provider/{id} - to view a specific user by with the given user id
  • GET /rest/provider/ - to view all users from excel data
  • GET /rest/ionservice/ - to view all users from MDR service
From a web browser you can access the services using the following links
  • provider/123 - to view the users from hardcoded / excel with id 123
  • provider/ - to list all users from docs/ hardcoded values
  • Users/ - to list all MDR users
